# springboot在线考试系统

------

### 技术栈：

> * 后端：springboot+shiro+JWT+MyBatis
> * 前端：Ajax+jQuery



### 部署方式

> 数据交互Rest接口
  前端：只要将examSystem文件夹放在nginx然后访问里面的静态文件即可。
  后端：将整个项目丢进去idea然后等项目导入完成然后导入项目的数据库到将mysql中再把项目的application.yml中的数据库账号密码配置下后启动即可。
  默认使用7800端口。
  前端的项目丢nginx或者apache里面，以文件打开方式浏览会出现cookie写不进去的问题。
